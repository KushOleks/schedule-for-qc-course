# Чеклісти для тестування

## Модуль: Загальна навігація та адаптивність

### Функціональність

- [ ] Перевірити, що кнопка **Home** відкриває головну сторінку
- [ ] Перевірити, що кнопка **Login** відкриває форму входу
- [ ] Перевірити, що можна вибрати **semester** зі списку
- [ ] Перевірити, що можна вибрати **Group** зі списку
- [ ] Перевірити, що можна вибрати **Teacher** зі списку
- [ ] Перевірити, що можна вибрати **Department** зі списку
- [ ] Перевірити, що після вибору параметрів розклад відображається
- [ ] Перевірити перемикання мови через прапорці

---

### Негативні сценарії

- [ ] Перевірити поведінку системи без вибору semester
- [ ] Перевірити що буде, якщо не вибрати Group / Teacher / Department
- [ ] Перевірити що станеться при швидкому натисканні кнопок
- [ ] Перевірити роботу системи при повільному інтернеті
- [ ] Перевірити відображення сторінки при дуже маленькій ширині екрана (<320px)

---

### UI/UX

- [ ] Перевірити відображення сторінки на мобільному екрані
- [ ] Перевірити відображення сторінки на планшеті
- [ ] Перевірити відображення сторінки на desktop
- [ ] Перевірити адаптацію інтерфейсу при зміні розміру вікна браузера
- [ ] Перевірити коректність відображення елементів у різних браузерах (Chrome, Firefox, Edge)
- [ ] Перевірити чи всі елементи інтерфейсу доступні та зрозумілі користувачу
